# Safa Website

Static landing page for Safa, an Islamic companion app for prayer times, Quran reading, Qibla, dhikr, Ramadan tools, Jumuah preparation, and daily Muslim guidance.

## Deployment

This repository is ready for GitHub Pages deployment through GitHub Actions.

The workflow at `.github/workflows/deploy.yml` publishes the site automatically whenever changes are pushed to the `main` branch.
